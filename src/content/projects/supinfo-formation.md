Au-delà des projets phares, le cursus SUPINFO couvre un socle large de modules techniques, dont une partie a directement nourri les projets présentés dans ce portfolio. Quelques modules par domaine :

## Cloud multi-fournisseur

- **Microsoft Azure** : Azure Fundamentals, Azure AI Fundamentals.
- **Google Cloud** : GCP Cloud Engineer, Google Cloud Cybersecurity.
- **Architecture cloud** : Développement d'Architecture Cloud (cf. [4ARCL](/projets/4arcl/)), APIs GraphQL (cf. [SchoolInc](/projets/4agql-proj/)).

## Réseau, Linux & sécurité

- **Réseau** : CCNA v7 - Switching, Routing and Wireless Essentials (Cisco).
- **Système** : Linux Professional Institute LPIC-2.
- **Sécurité applicative** : Sécurité des Applications, Droit des données personnelles (RGPD).

## Développement & mobile

- **Web** : Développement Web React avancé, Développement API Node.js, C#/ASP.NET, Développement Microservices (cf. [TicketApp](/projets/4webd-ticket/)).
- **Mobile** : Développeur Android, Développement d'applications hybrides Ionic/React.
- **Conteneurs & CI** : Docker (cf. [3DOKR](/projets/3dokr/)), Tests et Intégration Continue, Observabilité.

## Données & gestion de projet

- **Données** : Bases de données NoSQL, Big Data Fundamentals.
- **Gestion de projet** : Méthode Agile, Stratégie & cahier des charges, Manager de Programmes, Managing Change.

## Pourquoi c'est pertinent

Ce socle explique la facilité à passer d'un environnement à l'autre (VMware chez Cloud Temple, AWS en certification personnelle, Azure et GCP en cursus) plutôt que d'être enfermé dans un seul écosystème cloud.
